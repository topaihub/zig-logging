# Zig 0.16 关键变更

## std.Io 接口（最重要的变更）

I/O 通过参数传递，后端可互换，应用代码不需要改。

```zig
// ✅ 0.16：Io 作为参数
fn app(io: std.Io) !void {
    try std.Io.File.stdout().writeStreamingAll(io, "Hello\n");
}

// main 中初始化后端
pub fn main(init: std.process.Init.Minimal) !void {
    var threaded: std.Io.Threaded = undefined;
    try threaded.init(gpa, .{});
    defer threaded.deinit();
    return app(threaded.io());
}
```

后端选择：
- `std.Io.Threaded` — OS 线程，CLI/桌面应用
- `std.Io.Evented` — io_uring(Linux)/GCD(macOS)，高并发（实验性）

## ArrayList 初始化

```zig
// ✅ 0.16
var list: std.ArrayList(u8) = .empty;

// ❌ 已移除
var list = std.ArrayList(u8){};
```

## 依赖管理

- 依赖存放在 `zig-pkg/`（项目根目录，替代 .zig-cache）
- `zig-pkg/` 加入 .gitignore
- 临时使用本地 fork：`zig build --fork=/path/to/dep`

## 增量编译

```bash
zig build --watch -fincremental
```

保存即编译，错误毫秒级反馈。

## 类型解析变更

- `*u8` 和 `*align(5) u8` 现在是不同类型（可隐式转换）
- 依赖循环错误信息显示完整循环路径
- 仅作为命名空间使用的 struct 不再触发字段分析
